## 测试模块
* 测试模块不单独编写函数进行实现，而是采用在各函数内部进行测试的方式，通过在`循环内生成随机数`并各自计算 `编写的函数值`与`python提供的内置方法`所计算的函数值进行`减法运算`，通过事先确定的`阈值`与`差值`进行比较来确定编写的函数是否正确及可用。用户可通过在界面上输入`想要测试的值`并通过`test`按钮进行相关测试，测试结果将输出到界面上。

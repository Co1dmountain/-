
# arcsin函数
* 反正弦函数：arcsin
* 反正弦函数是反三角函数的一种。
* 定义域：[-1,1]
* 值 域：[-π/2,π/2]
* 奇偶性：奇函数
* 周期性：不是周期函数
* 单调性：[-1,1]单调递增

# arctan函数
* 反正切函数：arctan
* 正切函数y=tanx在开区间（x∈(-π/2,π/2)）的反函数，记作y=arctanx 或 y=tan-1x，叫做反正切函数。它表示(-π/2,π/2)上正切值等于 x 的那个唯一确定的角，即tan(arctan x)=x，反正切函数的定义域为R即(-∞，+∞)。
* 反正切函数是反三角函数的一种。
* 定义域：R
* 值 域：(-π/2,π/2)
* 奇偶性：奇函数
* 周期性：不是周期函数
* 单调性：（－∞，﹢∞）单调递增
